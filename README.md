##

Clone o repositório:
```bash
git clone https://github.com/brunomat480/coffee-delivery.git
```

Baixe as dependencias:
```bash
yarn
```
ou
```bash
npm i
```
##

inicie a aplicação:
```bash
yarn dev
```
ou 
```bash
npm run dev
```
